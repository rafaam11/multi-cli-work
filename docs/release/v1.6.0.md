# v1.6.0 — Git worktree 관리와 사이드바 확장

## 변경 사항

- Git의 실제 worktree 목록을 기준으로 외부에서 만든 worktree까지 자동 발견·동기화
- 새 브랜치, 기존 로컬 브랜치, 원격 추적 브랜치용 worktree 생성 지원
- Git 프로젝트를 `프로젝트 → 메인/worktree → 세션` 계층으로 표시하고 접힘·선택 상태 복원
- branch/HEAD, 변경 파일 수, locked·missing·prunable 상태와 프로젝트별 동기화 경고 표시
- worktree 동기화, 명시적 Fetch, unlock, stale 관리정보 정리, dirty 확인을 포함한 안전 제거 제공
- Quick Open, 상세 화면, 파일·Git 패널이 동일한 작업공간 선택을 공유
- Windows와 Linux 경로 비교 및 Git porcelain 파서 보강

## 릴리스 자산

- `Multi-CLI-Work-Setup-1.6.0.exe`
- `Multi-CLI-Work-1.6.0-linux-x64.deb` / `.AppImage`
- `latest.yml`, `latest-linux.yml` 및 생성된 blockmap

## 검증

- 단위·렌더러 테스트, TypeScript typecheck, 프로덕션 빌드 통과
- 실제 임시 Git 저장소를 사용한 외부 worktree 발견·ID 재사용·브랜치 생성·안전 제거 검증
- Electron E2E에서 외부 worktree 발견부터 PTY 세션 실행, 변경 표시, 제거까지 검증
